status

Show bot status

Displays the active cogs, when the bot was started and the uptime.

Usage

+status

checkpings

Check recent pings

Displays a list of the 10 most recent times you have been pinged (in order), who pinged you and when. Only logs pings while bot is active.

Usage

+checkpings

addtask

Set a task for a class

Select the class (role), the due date and the description of the task. The class gets reminded with a ping the day before it is due.

Usage

/addtask [role] [due date] [description]

removetask

Delete a task

Select the class (role) or leave blank to see all classes. A list of tasks is displayed; you can choose one to delete, or cancel.

Usage

/removetask [role]

removealltasks

Delete all tasks for a class

Select the class (role) or leave blank for all classes. A list of the tasks that will be deleted is displayed. It will ask you for confirmation; if you confirm, all shown tasks will be removed.

Usage

/removealltasks [role]

tasklist

View list of tasks for a class

Select the class (role) or leave blank for all classes. All of that class's tasks will be displayed.

Usage

/tasklist [role]

mytasks

View a list of your tasks

A list of all tasks for all classes you are in is displayed.

Usage

/mytasks

astronaut

Plays "Astronaut in the Ocean"

Connects to the voice channel you are in and plays "Astronaut in the Ocean". Disconnects when the song is finished.

Usage

+astronaut

laugh

Laugh

The bot laughs.

Usage

+laugh

roast

Insult another user

Selects a random insult and hurls it at the chosen user.

Usage

+roast [user]

thank

Thank someone

The bot thanks the user. You can also add "for" and a reason.

Usage

+thank [user] [for reason]
